# MacStream v0.5.0

MacStream v0.5.0 adds presenter composition for screen-focused livestreams.
Your display or selected window can fill the canvas while a movable, native
person cutout stays over the content.

## Highlights

- Switch Screen + Webcam between existing framed layouts and Cutout output.
- Place the presenter on the left, right, top, or bottom, or position it
  manually.
- Adjust presenter scale independently from webcam crop, zoom, and pan.
- Keep canvas padding and image or color backgrounds available around the
  full-screen content.
- Apply layout changes to active recording and RTMP output without restarting
  capture.

## Performance And Fallback

- Person segmentation uses Apple's native Vision framework with fast quality.
- Segmentation runs asynchronously on a dedicated serial queue, processes at
  most 12 frames per second, and coalesces pending camera frames.
- Capture, audio, composition, and RTMP delivery never wait for Vision.
- Mattes must be both recently processed and close to the current camera
  presentation time.
- When segmentation is unavailable, delayed, stale, or fails, MacStream keeps
  the stream running with a framed webcam overlay.

## Preview And Output

The setup preview uses the same placement geometry and shows a framed fallback
before real capture. During recording or RTMP publishing, the program preview
shows the same composited frames sent to the recorder and streaming
destinations, including any active cutout or framed fallback.

## Current Limits

MacStream remains a personal-use prototype. Validate important broadcasts with
a private or test destination first. Native camera-effect controls, video
editing, transcription, and other AI-assisted features remain deferred.
